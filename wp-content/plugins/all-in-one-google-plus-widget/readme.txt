== Description ==

This simple WordPress widget will add Google like button, Google Follow me button for user profile and Share page button for any of your website URL.There are many options which are configurable in admin widget area.Its easy to setup.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

For more details,
